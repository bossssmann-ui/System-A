# Telegram Voice Bot → Obsidian Inbox

> Спека для Builder Agent. Проект: бот, принимающий голосовые/текст/фото от собственника и складывающий в vault.

## Цель
Свести трение капчура к нулю: открыл Telegram → нажал микрофон → сказал → отправил → через 10–30 секунд запись в vault.

## Функциональность

### MVP (первая версия)
- [ ] Принимает голосовые сообщения от одного конкретного Telegram user_id (собственника).
- [ ] Транскрибирует через Whisper API (русский + английский, мульти-язык).
- [ ] Добавляет запись в файл `90_Inbox/YYYY-MM-DD.md` формата:
  ```
  ## HH:MM — voice
  {транскрипция}
  
  > 🔗 voice_{message_id}.ogg
  ```
- [ ] Подтверждает получение через реакцию ✅ в Telegram.
- [ ] При ошибке транскрипции — реакция ⚠️ + лог.

### V2 (после MVP)
- [ ] Принимает текстовые сообщения → сразу в inbox.
- [ ] Принимает фото → OCR при необходимости, файл в `90_Inbox/attachments/`, запись в inbox.
- [ ] Команда `/find <запрос>` — поиск по vault.
- [ ] Команда `/daily` — пересылает текущий Daily Note.
- [ ] Команда `/last` — последние 5 записей в inbox.
- [ ] Тегирование при записи через ключевые слова ("обещание", "идея", "задача").

### V3 (роскошь)
- [ ] AI-классификация: бот сам решает, что это — задача / обещание / идея / факт — и предлагает положить в соответствующее место.
- [ ] Авто-создание карточек людей/клиентов при упоминании.

## Технический стек

| Компонент | Выбор | Почему |
|---|---|---|
| Язык | Python | Telethon / aiogram + OpenAI SDK зрелые |
| Telegram | aiogram 3.x | современный, асинхронный |
| STT | OpenAI Whisper API | качество, поддержка русского |
| Storage | зависит от решения A/B/C в `Mobile_Capture_Plan.md` | |
| Хостинг | Hetzner CPX11 (€4/мес) или Fly.io free tier | надёжнее Workers для background-задач |
| Конфиг | `.env` с ключами | |

## Что нужно от собственника до старта разработки
1. Решение по вилке (GitHub / iCloud+Mac / Dropbox).
2. Ник и @ будущего бота.
3. Доступ к OpenAI API (ключ) — либо регистрируем новый аккаунт с привязкой к компании.
4. Telegram user_id (один, пока только для него).

## Безопасность
- Белый список Telegram user_id: только собственник + (опционально) РОП.
- Токен бота и OpenAI ключ — только в `.env`, никогда в vault.
- Логи без транскрипций — только метаданные (время, длительность).

## Оценка трудозатрат
- MVP: 6–10 часов разработки.
- V2: +5–8 часов.
- Стоимость запуска: ~$5/мес.
