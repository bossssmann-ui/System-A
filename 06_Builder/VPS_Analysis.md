## Подводные камни

### Верификация и оплата

| Провайдер | Требования к физ.лицу |
|---|---|
| Reg.ru | Базовая верификация номера телефона |
| RuVDS | Базовая верификация телефона |
| Beget | Email + телефон |
| TimeWeb | Может требоваться при крупных платежах |
| Selectel | Паспорт + адрес для физ.лиц |

**Если не хочется бюрократии** → RuVDS или Beget.

### Поведение при скрапинге

- **RuVDS** — явно поддерживает парсинг в политике, DDoS-защита включена.
- **Reg.ru** — официально не запрещает, но риск авто-блокировки при всплеске трафика. **Ставить sleep 2–5 сек между запросами.**
- **Beget** — дружелюбен, но политика не публикуется явно.
- **TimeWeb / Selectel** — более консервативны. Перед запуском парсера согласовать с поддержкой.

### Скрытые расходы

- TimeWeb повысил цены +8–20% с 15.12.2025.
- Selectel повысил цены +8–9% с 15.01.2026.
- RuVDS и Reg.ru — самые прозрачные на текущий момент.

### Защита тендерных площадок

Некоторые крупные площадки используют Cloudflare. Playwright в headless-режиме иногда блокируется. Решения:
- `headless=False` (с виртуальным дисплеем Xvfb на сервере)
- Прокси (RuVDS продают отдельно)
- Имитация реального браузера (User-Agent, Viewport, задержки между действиями)

## Финальный выбор

**Для старта рекомендую RuVDS VPS-2 Linux (249 ₽/мес).**

Логика:
1. 3-дневный тест → если парсер работает на нём, остаёмся (97 ₽/мес при годовой оплате).
2. Если возникнут проблемы (блокировки, нестабильность) → за 2–3 часа мигрируем на Reg.ru.
3. Suммарная экономия vs Reg.ru: ~6 000 ₽/год при годовой оплате RuVDS.

## Чек-лист перед запуском парсера на VPS

- ☐ Заказать VPS у выбранного провайдера
- ☐ Установить Ubuntu 22.04 LTS или Debian 12
- ☐ Поставить Python 3.10+, Playwright, Chromium
- ☐ Локально протестировать парсер на одной площадке
- ☐ Добавить sleep 2–5 сек между запросами
- ☐ Установить нерекламный User-Agent
- ☐ Cron-job: `0 9 * * * /usr/bin/python3 /path/to/parser.py`
- ☐ Anthropic API ключ через переменную окружения, не в коде
- ☐ Логирование в файл + отправка ошибок в Telegram
- ☐ Тестовый запуск в ночное время
