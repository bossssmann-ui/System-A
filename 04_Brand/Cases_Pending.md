# Кейсы — материалы для КП и сайта

> Список выполненных перевозок, по которым у Романа есть фотографии и контекст. Заполняется по мере, когда Роман присылает материалы.
>
> **Назначение**: будущие коммерческие предложения, страница "Кейсы" на сайте Тихоокеанская звезда, посты в WhatsApp/Telegram.

## Зачем это критично

Тендерные заявки и продажи новым клиентам — это **доказательная игра**. Голословное "мы возим грузы" слабее, чем "мы перевезли проект УСГ Магадан → Приморье на 6 млн ₽, мультимодал, отзыв заказчика". Один сильный кейс закрывает 3 первых звонка.

## Шаблон карточки кейса

Когда Роман присылает фото — заводим под каждый кейс отдельную карточку по формату:

```
### [Название] — [одна строка-сутевик]

- Заказчик: ...
- Период: ...
- Маршрут: ... → ...
- Тип груза: ...
- Транспорт: ... (виды)
- Сумма: ... ₽
- Сложности и как решили: ...
- Результат: ... (отзыв, повторный заказ, рекомендация)
- Фото: [ссылки или прикреплённые файлы]
```
