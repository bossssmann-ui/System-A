# 90_Inbox — всё новое сначала сюда

> Это главный файл системы с точки зрения личной памяти. Здесь не должно быть красиво. Сюда просто **падает всё**.

## Железное правило
**Не думай, куда класть мысль. Клади сюда. Разберём потом.**

Попытка классифицировать на лету = пропущенная мысль. Это запрещено.

## Как капчурить с разных устройств

### С iPhone (Obsidian mobile, главный канал)
1. Открыть Obsidian.
2. Открыть `90_Inbox/Quick.md`.
3. Добавить строку. Закрыть.

Если тянет долго — открой просто сегодняшний **Daily Note** (⌘+D или иконка) и сбрось туда в раздел "Inbox дня". Результат тот же: Memory Curator всё равно всё подметёт в понедельник.

### Запасной канал — диктофон iPhone
Если за рулём / на ходу / руки заняты — записывай голосовое **в свой личный чат в Telegram** (Saved Messages) или в заметку. В понедельник разберём на месте.

### Запасной канал — этот чат (AI-помощник)
Можно просто написать мне "запиши в inbox: ..." — я положу запись в `90_Inbox/Quick.md` с датой.

### Будущий канал — Telegram-бот (Builder Agent)
Отдельная задача. Бот возьмёт голос → расшифровку → добавит строку в `90_Inbox/YYYY-MM-DD.md`. См. `99_System/Mobile_Capture_Plan.md`.

## Что НЕ делать

- ❌ Не пытайся разложить мысль по правильным папкам в момент записи.
- ❌ Не оформляй, не форматируй — одна строка, свободный текст.
- ❌ Не удаляй ничего из inbox самостоятельно без разбора.
- ❌ Не накапливай больше 7 дней — разбирай минимум раз в неделю.

## Как разбирать (Memory Curator, раз в неделю, 15–20 минут)

Для каждой строки — одно из 4:
1. **Мусор** → удалить.
2. **Задача** → в задачник (текущий раздел / приоритет).
3. **Факт / знание** → в соответствующую папку (01–06, 10, 20, 30).
4. **Обещание** → в `10_People/Commitments.md`.

После разбора `Quick.md` должен стать пустым.

## Триггеры для разбора
- Понедельник 09:00 — основной разбор.
- Любой момент, когда Quick.md перевалил за 50 строк — разобрать немедленно.
